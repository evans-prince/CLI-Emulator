Every SimpleRISC program needs a .main: label -- execution starts there. Registers are r0-r13, plus sp (r14, stack pointer) and ra (r15, return address). .print r0, r1 prints registers to the Output panel; .encode <instruction> prints its 32-bit machine encoding. Labels + b/beq/bgt/call/ret give you branches, loops, and function calls.
